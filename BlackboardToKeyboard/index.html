Aquí va a ir contenido(1).

Actualmente lo que se quiere lograr en la página, es poder añadir el mensaje de indicación desde la página y recién enviar la recomendación o indicación por whatsapp. Para eso se necesita aprender javascript, usar HTML para crear un formulario y extrarer cada caracter introducido en el input del formulario, y adecuarlo a la formula: texto,espacio, texto; llevado a URL, siendo texto,%F,texto. Además es necesario realizar correciones respecto al responsive design, entre otras cosas más.
